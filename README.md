# billmainlandnelson.com
My website, built with [Hugo](https://github.com/gohugoio/hugo) and deployed on Github Pages

## Development

### Update submodules
```bash:
git submodule init
git submodule update
```

### Make your changes
* Create branch and checkout
* Test locally 
```bash:
Hugo server
```
* Push to a branch
* Make a pull request

## Deployment
* Merge pull request (push to Master)
